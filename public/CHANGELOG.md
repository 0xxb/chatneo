### v0.1.0

- 支持 Ollama 本地模型及 28+ 云端服务商（OpenAI、Anthropic、Google、硅基流动、智谱、DeepSeek 等）
- SSE 流式对话，支持扩展思考、图片发送、消息编辑与重新生成
- 多模型同时回答，对比不同模型的输出
- 知识库（RAG），对话中引用本地文档，支持文档分段与向量检索
- 语音输入（STT）：本地 sherpa-onnx 离线识别
- 语音输出（TTS）：本地 sherpa-onnx + OpenAI 兼容 API 双通道，支持模型下载管理
- 插件系统：网页搜索、URL 读取、天气、计算器、代码运行、文件读取、翻译等
- 自动生成对话标题
- 自定义系统提示词，可按对话挂载
- Markdown / 代码高亮 / LaTeX / Mermaid 渲染
- SQLite 本地存储，数据完整持久化
- WebDAV 云备份与恢复，支持定时自动备份
- 对话导出（Word），包含图片与附件
- 主题切换、强调色、字体、代码高亮主题
- 自定义键盘快捷键
- 模型收藏与按服务商分组管理
- macOS 原生毛玻璃效果、系统托盘、开机自启
